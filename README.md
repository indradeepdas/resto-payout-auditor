# Resto Payout Auditor

A web tool that audits restaurant payout reports from delivery platforms (Uber Eats, Wolt, Deliveroo, etc.) and detects discrepancies in commissions, missing orders, and payout leakage.

## Features
- Upload payout CSV from delivery platform
- Upload POS export
- Automatic reconciliation
- Commission validation
- Leakage detection
- Export discrepancy report
- PostHog observability

## Tech Stack
- React
- Vite
- JavaScript
- PostHog Analytics
